    ## Description <a name="description"></a>
    
    This project uses Node and Javascript to allow the user to enter in basic information about their project to quickly format a README file.
    
    <hr>
    
    ## Installation Instructions <a name="install"></a>
    
    The user will need a recent version of Node.js and inquirer from npm. The program will start when the user runs the program in the terminal
    
    <hr>
    
    ## Usage <a name="usage"></a>
    
    Type node index.js in the terminal to start the README generator 
    
    check out this link for a video on how to run the README generator and see it in action.
